# Nexus | Agentic Intelligence

A premium, immersive Agentic AI interface featuring real-time AI responses, image and PDF support, Markdown rendering, and code highlighting, all streaming live via Nvidia's high-performance APIs.

## Features
- **Fluid Response Streaming**: See text type out seamlessly using Server-Sent Events (SSE).
- **Dark-Mode Glassmorphism**: High-quality UI design with deep glowing panels and seamless interactions.
- **Multimodal Uploads**: Securely load text directly from PDFs or parse images.
- **Developer Ready**: Live parsing for Markdown and Highlight.js formats perfectly sizes technical queries.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone <URL>
   cd AgenticAI
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the backend:
   ```bash
   python app.py
   ```
4. Access the workspace at `http://localhost:5000`
